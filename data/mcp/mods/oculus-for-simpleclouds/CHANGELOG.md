# CHANGELOG

## 1.0.0
- Adds a Forge compatibility layer for Simple Clouds with Oculus shader rendering.
- Forces the shader-aware Simple Clouds pipeline when shaders are active.
- Exposes cloud, weather, and storm-related uniforms to supported shader integrations.
- Registers the Simple Clouds render target with the Oculus sky bridge when available.
- Adds a file-based cloud dimension whitelist at `config/oculus_for_simpleclouds/cloud_dimension_whitelist.json`.
- Documents Atmospheric Shaders as the only officially supported shader pack.
- Distant Horizons support remains experimental.
